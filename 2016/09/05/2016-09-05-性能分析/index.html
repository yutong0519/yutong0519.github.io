性能分析

c++ 程序性能分析

环境准备

安装 graphviz

sudo yum install graphviz.x86_64

安装 gprof2dot

sudo pip install gprof2dot

在 makefile 中增加 -pg 参数

导出图片

gprof 程序名 gmon.out|gprof2dot -s | dot -Tpng -o output.png

gprof2dot 后面带参数-s 是简化函数名
